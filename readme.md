# Web chat - Socket.io

O objetivo desse repositório foi criar um chat usando o Socket.io, que permite uma comunicação assíncrona entre servidor e cliente. Pretendo ainda fazer um projeto parecido, mas criando um servidor completo com banco de dados e um frontend com react.

O projeto foi baseado no vídeo "[Implementando WebSockets no Node.Js](https://www.youtube.com/watch?v=xEpE7DSOvVw)", feito pela Rocketseat.

Tecnologias usadas:

- express
- socket.io
